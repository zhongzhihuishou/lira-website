**版本：** V1.0  
**创建时间：** 2026-03-12  
**应用场景：** 智能回收机边缘计算

---

## 📋 方案概述

**核心选型：** RK3568 作为智能回收机边缘计算主芯片

**选择理由：**
- ✅ 成本低（¥150-400 vs Jetson Orin NX ¥3,000-5,000）
- ✅ 功耗低（3-5W vs 10-25W）
- ✅ 国产自主可控
- ✅ 性能足够（YOLOv8n 20-30 FPS）
- ✅ 被动散热，无噪音

---

## 🎯 RK3568 核心参数

| 参数 | 规格 |
|------|------|
| **CPU** | 4 核 Cortex-A55，最高 2.0GHz |
| **GPU** | Mali-G52 2EE，支持 OpenGL ES 3.2 |
| **NPU** | 1 TOPS (INT8)，支持 YOLO/SSD/MobileNet |
| **内存** | 2GB/4GB/8GB LPDDR4/LPDDR4X |
| **存储** | eMMC 5.1 / TF 卡 / SATA SSD |
| **视频** | 4K@30fps 解码，1080P@30fps 编码 |
| **摄像头** | 支持 MIPI-CSI 2 路 / USB 摄像头多路 |
| **网络** | Gigabit 以太网 / WiFi / 4G/5G |
| **接口** | USB 3.0, USB 2.0, GPIO, I2C, SPI, UART |
| **功耗** | 3-5W（典型） |
| **工作温度** | -20°C ~ 70°C |

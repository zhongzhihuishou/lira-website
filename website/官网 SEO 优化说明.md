### 2. Schema.org 结构化数据

#### Corporation 结构化数据
```json
{
    "@type": "Corporation",
    "name": "莱拉人工智能（吉林省）有限公司",
    "alternateName": "Lira Intelligence",
    "url": "https://www.lira-ai.cn",
    "industry": "Industrial AI & Robotics"
}
```

#### FAQ 结构化数据
```json
{
    "@type": "FAQPage",
    "mainEntity": [
        {
            "@type": "Question",
            "name": "Lira 的核心产品是什么？",
            "acceptedAnswer": {...}
        },
        ...
    ]
}
```

**SEO 收益：**
- Google 富媒体搜索结果（Rich Snippets）
- 知识图谱展示
- FAQ 直接展示在搜索结果
